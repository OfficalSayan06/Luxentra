## 📖 Overview

LUXENTRA is a modern responsive hotel booking website designed to provide a seamless online booking experience. The website features elegant layouts, intuitive navigation, and a fully responsive design that adapts across desktop, tablet, and mobile devices.

Built as a front-end project, it demonstrates responsive web design principles, clean code organization, and modern UI development using Bootstrap.

---

## ✨ Features

- Responsive Design
- Modern Hotel Booking Interface
- Multi-Page Website
- Room Showcase
- Hotel Services
- Booking Form
- About & Contact Pages
- Bootstrap Components
- Smooth Navigation
- Cross Browser Compatibility
